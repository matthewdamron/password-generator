# Password Generator

## Description

This website will randomly generate a password bassed on user input.
1. Password length 8 to 128 characters long
2. Lowercase characters
3. Uppercase characters
4. Numeric characters
5. Special characters

## Criteria

GIVEN I need a new, secure password<br>
WHEN I click the button to generate a password<br>
THEN I am presented with a series of prompts for password criteria<br>
WHEN prompted for password criteria<br>
THEN I select which criteria to include in the password<br>
WHEN prompted for the length of the password<br>
THEN I choose a length of at least 8 characters and no more than 128 characters<br>
WHEN prompted for character types to include in the password<br>
THEN I choose lowercase, uppercase, numeric, and/or special characters<br>
WHEN I answer each prompt<br>
THEN my input should be validated and at least one character type should be selected<br>
WHEN all prompts are answered<br>
THEN a password is generated that matches the selected criteria<br>
WHEN the password is generated<br>
THEN the password is either displayed in an alert or written to the page<br>
